# 闭包

>闭包就是能够读取其他函数内部变量的函数  
MDN: 闭包是函数和声明该函数的词法环境的组合。

```
# 访问函数内部变量
function f1(){
  n=999;
  function f2(){
    alert(n);
  }
  return f2;
}
var result=f1();
result(); // 999
```



## 缺点
- 滥用会导致内存泄漏： 解决方法时，在退出函数之前，将不使用的局部变量全部删除。
