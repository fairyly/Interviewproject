## 2.找出一个数组中数值元素最大值

```
1.es6拓展运算符...
Math.max(...arr)

2.es5 apply(与方法1原理相同)

Math.max.apply(null,arr)  
or  
Math.max.call(null,...[1,5,6,9])

3.for循环

let max = arr[0];
for (let i = 0; i < arr.length - 1; i++) {
    max = max < arr[i+1] ? arr[i+1] : max
}

4.数组sort()

arr.sort((a, b) => {
    return  b - a
})
arr[0]

5.数组reduce

arr.reduce((a, b) => {
    return a > b ? a : b}
)

```
