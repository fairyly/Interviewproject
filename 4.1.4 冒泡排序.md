# 冒泡排序

- 时间复杂度：O(n2)
- 空间复杂度：O(1)

>思想：比较相邻的元素。如果第一个比第二个大，就交换它们两个；

```
function sort(arr) {

  for(var i=0;i<arr.length-1;i++){
    for(var j=i+1;j<arr.length;j++){
      if(arr[i]>arr[j]){//从arr[0]开始比较大小
        var tem=arr[i];
        arr[i]=arr[j];
        arr[j]=tem
      }
    }
  }
  return arr;
}
```
